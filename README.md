<h1>CHATBOT - with Spring AI- React - Vaadin - RAG - Llama-Open AI</h1>

<p>Ce projet vise à concevoir et développer un micro-service de chatbot intelligent basé sur l’architecture RAG (Retrieval-Augmented Generation). Il s’appuie sur des technologies modernes de traitement du langage naturel (NLP) et de recherche d’information afin de proposer un système capable de générer des réponses pertinentes, précises et contextualisées. Ce type de chatbot est particulièrement adapté aux applications nécessitant une interaction naturelle avec les utilisateurs et une exploitation efficace d’une base de connaissances spécifique.</p>
