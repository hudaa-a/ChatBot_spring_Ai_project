# Chatbot Intelligent – Microservice RAG

## Description
Ce projet consiste à développer un **microservice de chatbot intelligent** en utilisant l’architecture **RAG (Retrieval-Augmented Generation)**.  
Le système combine des technologies modernes de **NLP (traitement du langage naturel)** et de **recherche d’information** pour fournir des réponses précises et contextuelles.  
Ce type de chatbot est idéal pour des applications nécessitant **une interaction naturelle** et un accès à une **base de connaissances spécifique**.

---

## Fonctionnalités
- **Chat en temps réel** avec l’utilisateur  
- **Recherche d’information** dans une base de données ou documents  
- **Réponses contextuelles** grâce à l’architecture RAG  
- **API REST** pour l’intégration avec d’autres services
